# Supply-chain-management-excel-dashboard
## Overview

This project is an interactive **Supply Chain Management Dashboard**
built in **Microsoft Excel**. It provides business insights into orders,
suppliers, warehouses, product categories, and monthly performance using
Pivot Tables, Pivot Charts, KPI Cards, Slicers, and Excel formulas.

## Dashboard Pages

-   On-Time Delivery %
-   Returned Orders %
-   Fulfilled Orders %

### 3. Warehouse Location Analysis
